Rock, Paper, Scissors Game

This game is designed to function inside of the Google Chrome browser console.

To play the game, you'll first need to open the console inside DevTools, per the following instructions:

  1. Right-click on the page, click on Inspect from the menu, then navigate to the tab.
  2. Alternatively, use the following shortcut:
    • For Mac: cmd + option + J
    • For Windows, Linux, ChromeOS: ctrl + shift + J

Once you're in the broswer console, type playGame() to play a game of rock, paper, scissors.
